# Stardust Sandbox Resources

This repository is intended to serve as a generic location for storing media content for use on the internet related to the game Stardust Sandbox, through reference links. These locations include **itch** and other services.

Here you will find promotional images, banners, icons, and other graphic materials intended for the promotion and external use of the project. The goal is to make these resources easily accessible to content creators, partners, and platforms that wish to promote or reference Stardust Sandbox.

> [!IMPORTANT]  
> This repository does not contain internal game assets such as sprites, models, or source code files. All resources used directly in the development of Stardust Sandbox are available exclusively in the project's official repository.

If you need specific promotional materials or have questions about the use of the content provided here, feel free to open an issue or contact the project team.
